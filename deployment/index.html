Deploying an AkashaCMS web site

  1. AkashaCMS - the powerful static file CMS to build your dream website
  2. Deploying an AkashaCMS web site

Once a website is rendered, the root_out directory contains the rendered website. Obviously, the goal is deployment to a public webserver.

One way to do this is

$ akashacms build
$ (cd out; scp -r . user@server.domain:example.com )

But we have a much easier way

$ akashacms build
$ akashacms deploy

There are two methods for deployment, rsync and ssh.

  • /deployment/index.html
    • Deployment using rsync
